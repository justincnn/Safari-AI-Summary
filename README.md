## English

A powerful browser extension that adds AI-powered page summarization functionality to Safari and other browsers. This userscript allows you to quickly generate concise summaries of any webpage using OpenAI's GPT model.

### Versions

This project provides three versions of the script to suit different needs:

1.  **Mobile Version (Recommended for Mobile)**: `mobile-ai-summary.user.js`
    *   Designed specifically for mobile devices (iOS/Android).
    *   Features a modern **Material Design 3 (Pixel Style)** interface.
    *   Includes Dark Mode support.
    *   Optimized touch interactions with a floating action button (FAB) and bottom sheet.
    *   [Install Mobile Version](mobile-ai-summary.user.js)

2.  **Safari Desktop Version**: `safari-ai-summary.user.js`
    *   Optimized for Safari on macOS.
    *   Clean floating UI.
    *   Draggable interface.
    *   [Install Safari Version](safari-ai-summary.user.js)

3.  **Chrome Desktop Version**: `chrome-ai-summary.user.js`
    *   Designed for Chrome/Edge/Tampermonkey on desktop.
    *   Includes additional features like OneDrive sync (experimental).
    *   [Install Chrome Version](chrome-ai-summary.user.js)

### Installation

1.  Install a userscript manager extension:
    *   **iOS (Safari)**: [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887)
    *   **Android (Kiwi/Firefox)**: [Tampermonkey](https://www.tampermonkey.net/)
    *   **Desktop**: [Tampermonkey](https://www.tampermonkey.net/)
2.  Click the installation link for your preferred version above.
3.  Configure your OpenAI API key in the settings panel (click the gear icon or right-click the robot icon).

### Configuration

Access the configuration panel to:
- Set your OpenAI API URL and API Key
- Customize the prompt used for summarization
- Select the AI model (e.g., gpt-3.5-turbo, gpt-4)
- Switch themes (Mobile version)
